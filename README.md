# Screenix Flatpak

Flatpak packaging for Screenix.

Screenix is proprietary software. The Flatpak manifest uses `extra-data` to
download the official release `.deb` from the public Screenix releases
repository during installation.

Current package source:
https://github.com/mathisdev7/screenix-releases/releases/download/v1.0.2/Screenix_1.0.2_amd64.deb
